# 🚀 快速开始指南

## 第一步：启动应用

### 方式 1：Web 版本（推荐用于开发）

```bash
cd tauri-md
npm run dev
```

然后在浏览器中访问：**http://localhost:1420/**

### 方式 2：Tauri 桌面版

```bash
cd tauri-md
npm run tauri dev
```

这将启动一个原生桌面应用窗口。

## 第二步：开始编辑

### 使用工具栏

应用顶部有一个功能丰富的工具栏，包含以下按钮：

#### 文本格式化
- **B** - 加粗文本
- **I** - 斜体文本
- **S** - 删除线
- **</>** - 行内代码

#### 标题
- **H1** - 一级标题
- **H2** - 二级标题
- **H3** - 三级标题

#### 列表
- **• List** - 无序列表
- **1. List** - 有序列表
- **☑ Task** - 任务列表

#### 块级元素
- **" Quote** - 引用块
- **{ }** - 代码块
- **―** - 水平分隔线

#### 其他
- **⊞ Table** - 插入 3x3 表格
- **↶** - 撤销
- **↷** - 重做

### 直接输入 Markdown

你也可以直接输入 Markdown 语法，编辑器会实时渲染：

```markdown
# 这是一级标题

这是一个段落，包含 **加粗** 和 *斜体* 文本。

- 列表项 1
- 列表项 2

\`\`\`javascript
console.log("Hello, World!");
\`\`\`
```

## 第三步：使用高级功能

### 代码块语法高亮

输入三个反引号 + 语言名称：

\`\`\`javascript
function hello() {
  console.log("Hello!");
}
\`\`\`

支持的语言：javascript, typescript, python, java, c, cpp, go, rust, php, ruby, swift, kotlin, bash, sql, json, yaml, html, css 等。

### 数学公式

#### 行内公式
使用单个 `$` 包裹：

```
质能方程：$E = mc^2$
```

#### 块级公式
使用双 `$$` 包裹：

```
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### 任务列表

```markdown
- [x] 已完成的任务
- [ ] 待办任务
```

点击复选框可以切换完成状态！

### 表格

使用工具栏的"⊞ Table"按钮插入表格，或手动输入：

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
```

## 第四步：文件操作

### 新建文档
点击菜单栏的"新建"按钮

### 打开文档
1. 点击"打开"按钮
2. 选择 .md、.markdown 或 .txt 文件
3. 文件内容将加载到编辑器

### 保存文档
1. 点击"保存"按钮
2. 选择保存位置
3. 文件将以 .md 格式保存

### 导出 HTML
1. 点击"导出 HTML"按钮
2. 生成的 HTML 文件将自动下载

## 💡 实用技巧

### 1. 快捷键
- **Ctrl + B** - 加粗
- **Ctrl + I** - 斜体
- **Ctrl + Z** - 撤销
- **Ctrl + Y** - 重做

### 2. 未保存提示
菜单栏会显示一个黄色的 **●** 表示文档有未保存的更改。

### 3. 实时预览
所有 Markdown 语法都会实时渲染，无需切换预览模式。

### 4. 表格调整
表格支持列宽调整，拖动列边界即可。

## 🎯 常见问题

### Q: 如何插入图片？
A: 目前版本使用标准 Markdown 语法：`![描述](图片URL)`

### Q: 支持哪些数学符号？
A: 支持完整的 LaTeX 数学语法，参考 [KaTeX 文档](https://katex.org/docs/supported.html)

### Q: 如何更改编辑器主题？
A: 当前版本使用固定主题，未来版本将支持主题切换。

### Q: 文档保存在哪里？
A: Web 版本使用浏览器下载功能，Tauri 版本将支持直接文件系统访问。
